# SBGroup - Hybrid Mobile Application

This project is a hybrid mobile application developed as part of a Bachelor's degree. It focuses on creating a cross-platform experience using web technologies integrated with native mobile functionalities.

## 📱 Project Overview

SBGroup (also referred to as **SBBerries** in documentation) is designed to provide a streamlined mobile interface for managing specific group or business data. The application is built using the **Apache Cordova** framework, allowing a single codebase to run on both Android and iOS.

### Key Features
- **Hybrid Architecture**: Built with HTML, CSS, and JavaScript.
- **Cross-Platform**: Support for Android and iOS via Cordova platforms.
- **Data Driven**: Structured backend logic based on a relational database model.
- **Responsive UI**: Wireframed for optimal mobile user experience.
