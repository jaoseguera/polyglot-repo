# Contactos Zamora

A mobile application built with **Ionic 3.0** and **Angular** designed to serve as a local business directory and event hub for Zamora, Michoacán.

## 📱 Project Purpose

This project was created as a learning exercise to master the Ionic framework. The application provides two main functionalities:
1. **Local Directory**: A repository of phone numbers and contact information for local businesses.
2. **Community Events**: A platform to share and view public events happening in Zamora.

## 🛠 Tech Stack

- **Framework**: [Ionic 3](https://ionicframework.com/docs/v3/)
- **Frontend**: Angular 4+, TypeScript, HTML5, SCSS
- **Platform Support**: Cross-platform (Android/iOS)
- **Deployment**: Includes build files for Android (.apk).

## 📂 Key Components

- `/src`: Contains the source code of the application (pages, providers, and templates).
- `config.xml`: Configuration for Cordova/Ionic features and app metadata.
- `Comandos.txt`: A reference file for useful CLI commands during development.
- `Contactos.apk`: A pre-built binary for testing on Android devices.
